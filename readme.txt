Add Dropdown List of Bangladesh District and Upazilla to the popular Contact Form 7 plugin.

== Description ==

Add Dropdown List of Bangladesh District and Upazilla to the popular <a href="https://wordpress.org/plugins/contact-form-7/" target="_blank">Contact Form 7 plugin</a>. 

<h4>Major features</h4>

<ul>
<li>Add a dropdown list of 64 districts of Bangladesh</li>
<li>Add a dropdown list of 492 Upazillas of Bangladesh</li>
<li>The Upazillas can be selected based on District</li>
<li>Both District and Upazillas are available in Bangla and English</li>
</ul>

== Installation ==

1. Upload the plugin or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen (Plugins > Installed Plugins).
3. Open Contact form 7 form editor.
4. There u go. You will find the option to add District and Upazilla list.
